# VSave

Download anything from the internet — videos, images, files, and more — directly to your Android device.

## About

This app lets you download content from various websites and links, all in one place. Simple, fast, and easy to use.

## Features

- Download videos, images, and files from links
- Simple and clean interface
- Manage and view your downloads
- Lightweight and fast

## Installation

1. Go to the [Releases](https://github.com/akramsts/vsave/releases) section of this repository and download the latest APK file.
2. Open the downloaded file on your Android device.
3. If prompted, allow installation from unknown sources.
4. Install and open the app.

## Requirements

- Android 7.0 or above
- Internet connection

## Built With

- [Kotlin](https://kotlinlang.org/) — Programming language

## Permissions

This app may request storage and internet permissions to download and save files to your device.
